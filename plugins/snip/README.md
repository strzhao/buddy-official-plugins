# snip — 文本片段速取

存常用文本片段（签名、地址、常用回复等），输入 `snip <关键词>` 立即取出并自动复制到剪贴板。参考 Alfred Snippets，零 LLM 秒回。

## 用法

```
snip <关键词>        # 精确命中 → 取出片段 + 自动复制到剪贴板
snip                 # 空查询 → 列出全部片段（选中可删除）
snip <模糊词>        # 模糊匹配 → 列出候选（选中可删除）
```

## 示例

预置片段 `sig` = `张三 13800138000`：

```
snip sig             # → 张三 13800138000（已自动复制）
snip si              # → 候选：sig: 张三 13800138000
snip                 # → 列出所有片段
```

## 动态占位符

片段内容支持以下占位符，取出时自动展开：

| 占位符 | 展开 | 示例 |
|---|---|---|
| `{date}` | 当天日期 YYYY-MM-DD | `2026-07-03` |
| `{time}` | 当前时间 HH:MM | `14:30` |
| `{clipboard}` | 当前剪贴板内容 | 取片段瞬间读取系统剪贴板 |

**未定义或畸形的占位符（如 `{nope}`、`{date`）原样保留**，不会报错。

### 不支持 `{cursor}`

Alfred Snippets 的 `{cursor}` 在 buddy launcher 中**不支持**（macOS 粘贴无法控制光标位置）。README 明示此限制。

## 添加 / 修改片段

snip 是**只读**插件（取/列表/删除）。添加和修改片段用配套的 `snip-mgr` 插件（自然语言）：

```
加个 sig 内容 张三 13800138000
把 sig 改成 李四 13900139000
```

## 删除片段

在 snip 候选列表中选中某条片段（带 🗑️ 标记）即可删除，会弹出二次确认。

## 数据存储

- 文件：`~/.buddy/snippets.json`
- 格式：JSON 数组，元素 `{keyword, content, created_at, updated_at}`
- keyword 唯一，仅允许字母数字 `_` `-`
- 原子写（临时文件 + rename），并发安全

## 依赖

- `jq`（JSON 处理）— 首次执行时 app 自动弹信任框 + `brew install`

## 限制

- `{cursor}` 占位符不支持
- keyword 白名单 `[A-Za-z0-9_-]`（防 shell 注入）
- content 长度上限 10000 字符
